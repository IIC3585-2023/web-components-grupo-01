## Stack

- Node 18.0.0
- TypeScript
- TailwindCSS
- Vite
- Lit

## How to run

### Install pnpm and dependencies

```bash
npm install -g pnpm
```

```bash
pnpm install
```

### Development Server

Start the development server on `http://localhost:5173`:

```bash
pnpm dev
```

### Production

Build the application for production:

```bash
pnpm run build
```

Locally preview production build:

```bash
pnpm run preview
```
